    Welcome

    Welcome to the NuGet wiki. These pages are primarily intended for those who wish to contribute to the NuGet project by submitting bug reports, suggesting new features, commenting on new ideas, or even submitting proposals.

    Please refer to the sidebar (on the right) for details on project management, contributing to NuGet, and documentation.

    If you are looking for more information on using NuGet, please visit our documentation and follow us on Twitter!
